# Audio assets — sourcing + licensing

Spec §11 lists 10 sounds. The audio system (`apps/webapp/src/audio/`) was
wired in Phase 5 and shipped silent for a long time — no files existed, so
every `audio.play()` was a no-op on every device.

**As of 2026-06-01 the SFX set is filled by procedural synthesis.** All
eleven one-shot cues are generated by [`scripts/generate_sounds.py`](../scripts/generate_sounds.py)
— pure-stdlib additive/subtractive synthesis tuned around C major, encoded
to stereo MP3 via ffmpeg. They are **100% original, no third-party license**
(no attribution needed, safe for commercial use). Background music is a
separate playlist of user-supplied tracks (see below). Regenerate / tweak:

```bash
python3 scripts/generate_sounds.py   # writes apps/webapp/public/sounds/*.mp3
```

To swap in higher-fidelity recordings later, drop replacements with the
same names into `apps/webapp/public/sounds/`. If you do, use only
royalty-free, license-clean sources and log them below:

- [Freesound](https://freesound.org) — filter for CC0
- [Pixabay Sound Effects](https://pixabay.com/sound-effects/) — Pixabay license, free to use
- [Mixkit](https://mixkit.co/free-sound-effects/) — free for commercial use
- [Zapsplat](https://www.zapsplat.com) — free with attribution / no-attribution paid

## Background music playlist

User-supplied tracks, re-encoded to **112 kbps stereo + loudness-normalized**
(`loudnorm I=-16`, metadata/art stripped) so volume is even across the set
and the total stays light (~16 MB for six tracks). Played as a **shuffled
sequence** (each streams once via html5, then the next — never the same one
twice in a row). Only the **host** plays the bed inside a room; guests
suppress it so co-located players don't stack tracks.

Per-track source links deep-link by **ISRC** to incompetech.com (Kevin
MacLeod's own site — the attribution domain named in the credit format
below). We use incompetech.com rather than filmmusic.io because the latter
migrated to ende.app and its per-track pages now 404; the ISRC URLs on
incompetech.com are the durable source. URL shape:
`https://incompetech.com/music/royalty-free/index.html?isrc=<ISRC>` (the
page resolves the track client-side, so it renders in a browser / Telegram
webview, not in a no-JS fetch). These are the links the in-app Credits
screen opens.

## Why the system is silent without assets

`AudioManager.play()` (in `apps/webapp/src/audio/index.ts`) creates each
`Howl` lazily on first use and registers `onloaderror` + `onplayerror`
handlers that swallow errors. The rest of the codebase remains
indifferent to whether sound files are present.
